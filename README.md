Shell script that sets Vinegar's studio channel reg value for you so studio loads alternative channels.
Allows arbitrary strings, but includes shorthands `-p` and `-sa` for `production` and `zserverauthoritycoreapi-earlyacces`.
I wrote this because I was too lazy to spend 10 minutes fixing it myself.

## Installation
Clone repo, cd into `vinegar-chswap`, and run `sudo cp vinegar-chswap /usr/bin`
